# Healthcare Data Orchestrator

A modern healthcare platform enabling secure patient data management, intelligent appointment scheduling, and medical record tracking. This project explores a microservices architecture for healthcare data systems, focusing on reliability and secure access.

## Key Features
*   Secure patient data management with JWT authentication.
*   Intelligent appointment scheduling system.
*   Centralized medical record tracking and access.
*   RESTful API built with a FastAPI microservices architecture.

## Tech Stack
FastAPI, Python, JWT, Pydantic, Uvicorn

## Getting Started
1.  Clone the repository: `git clone https://github.com/zoreanuj/healthcare-data-orchestrator.git`
2.  Install dependencies: `pip install -r requirements.txt`
3.  Run the development server: `uvicorn main:app --reload`
